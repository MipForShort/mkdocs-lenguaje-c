
# 5.- Estructuras de Control de Flujo

Un programa se basa en un conjunto de **sentencias** o **instrucciones**. Estas pueden contener un tipo especial de sentencias que dan control sobre objetos o datos del programa.

En C, las sentencias terminan con `;` (punto y coma). Estas se agrupan en **bloques** encerrados entre llaves `{ }`.

!!! note "Agrupación de sentencias"
    Las sentencias que controlan el flujo pueden contener otras sentencias dentro de ellas. Esto se llama **anidación**.

---

## if else

`if` se utiliza para ejecutar una sentencia (o grupo de ellas) dependiendo del valor de una condición.  

**Sintaxis:**

```c
if (condicion)
{
    sentencias 1;
}
else
{
    sentencias 2;
}
```

!!! tip
    El bloque `else` es opcional, pero se recomienda utilizar llaves `{}` incluso en sentencias simples para mejorar la legibilidad.

Ejemplos:

```c
if (1)
{
    printf("Hola");
}
else 
{
    printf("Adios");
}

/* Par o Impar */
if (n % 2 == 0)
{
    printf("Par");
}
else
{
    printf("Impar");
}
```

## switch

La estructura `switch` permite ejecutar distintas ramas en función de un valor entero.  

**Sintaxis:**

```c
switch (expresion_entera)
{
    case const1:
        sentencias1;
        break;
    case const2:
        sentencias2;
        break;
    ...
    default:
        sentencias_por_defecto;
        break;
}
```

!!! warning
    La expresión del `switch` **nunca debe ser `float`**. Solo se permite `int` o `char`.  
    Además, **siempre incluye `break`** para evitar ejecución no deseada de otros casos.

Ejemplo 1:

```c
switch (nota)
{
    case 'A':
        printf("Sobresaliente");
        break;
    case 'B':
        printf("Notable");
        break;
    case 'C':
        printf("Aprobado");
        break;
    case 'D':
        printf("Suspendido");
        break;
    default:
        printf("Nota no valida");
        break;
}
```

Ejemplo 2:

```c
switch (nota)
{
    case 10:
    case 9:
        printf("Sobresaliente");
        break;
    case 8:
    case 7:
        printf("Notable");
        break;
    case 6:
    case 5:
        printf("Aprobado");
        break;
    case 4:
    case 3:
        printf("Suspendido");
        break;
    case 2:
    case 1:
    case 0:
        printf("Muy deficiente");
        break;
    default:
        printf("Nota no valida");
        break;
}
```

## else if

Se usa cuando se requiere evaluar múltiples condiciones en orden.  

**Sintaxis:**

```c
if (condicion_1)
{
    sentencias1;
}
else if (condicion_2)
{
    sentencias2;
}
else
{
    sentencias_default;
}
```

Ejemplo:

```c
#include <stdio.h>
#include <stdlib.h>

int main(void)
{
    float nota;
    printf("Inserte la nota:\n");
    scanf("%f", &nota);

    if (9 <= nota && nota <= 10)
        printf("Sobresaliente\n");
    else if (7 <= nota && nota < 9)
        printf("Notable\n");
    else if (5 <= nota && nota < 7)
        printf("Aprobado\n");
    else if (0 <= nota && nota < 5)
        printf("Suspendido\n");
    else
        printf("Nota no valida\n");
}
```

## while y do while

Se utilizan para repetir instrucciones mientras se cumpla una condición (**bucles**).

**while:**

```c
while (condicion)
{
    sentencias;
}
```

**do while:**

```c
do
{
    sentencias;
}
while (condicion);
```

!!! note
    En `do while` el bloque se ejecuta **al menos una vez**, incluso si la condición es falsa al inicio.

Ejemplo:

```c
#include <stdio.h>

int main(void)
{
    int i = 1;

    while (i < 10)
    {
        printf("Hola %d\n", i);
        i++;
    }

    do
    {
        printf("Hola %d\n", i);
        i--;
    }
    while (i >= 1);
}
```

## for

Estructura común para bucles con contador.

**Sintaxis:**

```c
for (inicializacion; condicion; variacion)
{
    sentencias;
}
```

Ejemplo:

```c
for (int i = 0; i < 10; i++)
{
    printf("Hola %d\n", i);
}
```

!!! warning
    Si omites las tres expresiones en el `for`, se genera un **bucle infinito**.

---

## break y continue

- `break`: Sale del bucle actual o de un `switch`.
- `continue`: Salta a la siguiente iteración del bucle.

Ejemplo de `break`:

```c
for (...)
{
    if (...)
    {
        break;
    }
}
```

Ejemplo de `continue`:

```c
for (int i = 0; i <= 100; i++)
{
    if (i % 2 == 0)
        continue; // Salta pares
    printf("%d\n", i);
}
```

## goto

Permite saltar a una etiqueta en el código.

**Sintaxis:**

```c
goto etiqueta;
/* ... */
etiqueta:
    sentencias;
```

!!! danger "Advertencia sobre goto"
    **No se recomienda usar `goto`**, ya que hace el código difícil de leer y mantener.  
    Su uso debe ser excepcional.
